# Private Encrypted Storage App

A secure cloud-based file storage system that allows users to upload, manage, and share files safely in the cloud. The application focuses on privacy, authentication, and efficient file management similar to modern cloud drives.

## Project Objective

Build a secure cloud storage application similar to Google Drive with authentication, file versioning, and sharing capabilities.

## Features

- Secure user authentication using OAuth 2.0
- Upload and store files in cloud storage
- File versioning support
- File sharing between users
- Encrypted file storage
- Simple and responsive web interface

## Tech Stack

**Frontend**
- React / HTML / CSS / JavaScript

**Backend**
- Node.js
- Express.js

**Cloud Storage**
- AWS S3 / Azure Blob / Google Cloud Storage

**Authentication**
- OAuth 2.0

## Installation

Clone the repository:

```bash
git clone https://github.com/omkarverse/private-encrypted-storage-app.git
```

Navigate to the project directory:

```bash
cd private-encrypted-storage-app
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```
